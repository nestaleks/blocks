# Colors category — premium business palettes

Раздел `colors` добавлен как отдельная категория в общем каталоге компонентов и содержит **20 example-палитр** для сайтов бизнес/премиум-класса.

## Что внутри
- 20 готовых палитр с отдельными превью и демо-сценой.
- В каждой палитре есть набор HEX-цветов (кнопка копирования и копирование по клику на swatch).
- Палитры ориентированы на популярные сочетания для B2B, fintech, legal, consulting, enterprise SaaS и luxury service брендов.

## Наиболее популярные premium-направления для бизнес сайтов
- **Navy + Gold + Ivory** (финансы, private banking, legal)
- **Graphite + Platinum + White** (enterprise/SaaS, консалтинг)
- **Indigo/Blue + Cool Gray + Cyan Accent** (технологии, B2B продукты)
- **Emerald + Charcoal + Warm Neutral** (ESG, healthcare, strategy)
- **Black/Onyx + Metallic Accent** (luxury и high-end сервисы)
