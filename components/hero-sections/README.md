# Premium Hero Sections — HTML CSS JS

Готовый мини‑проект с коллекцией адаптивных premium hero-секций.

## Что внутри

- `index.html` — все секции на одной странице
- `css/style.css` — вся стилизация и адаптив
- `js/main.js` — переключатель темы и анимация появления при скролле

## Секции

1. SaaS / AI dashboard
2. Luxury / interior collection
3. Creative agency
4. Premium product
5. Real estate
6. Travel / resort
7. Finance / dashboard
8. Minimal architecture

## Как запустить

Откройте `index.html` в браузере.

Для удобной локальной разработки можно запустить через VS Code Live Server или через терминал:

```bash
cd premium-hero-sections
python -m http.server 8000
```

Потом открыть:

```text
http://localhost:8000
```

## Как использовать в своих проектах

1. Скопируйте нужную секцию из `index.html`.
2. Перенесите связанные CSS-классы из `style.css`.
3. Замените тексты, ссылки и изображения.
4. Настройте цвета в `:root`.

Изображения используются через Unsplash URL. При необходимости скачайте свои изображения в папку `img/` и замените `src`.
